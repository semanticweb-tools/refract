# Global Action

**Definition type:** OWL Class \
**Fragment identifier:** `fract:GlobalAction` \
**URI:** `https://w3id.org/refract#GlobalAction`

A macro-level refactoring action that operates on an ontology as a whole, rather than on individual ontology elements.

Global actions include:

* [AddNamespacePrefix](../RefactoringActions/AddNamespacePrefix)
* [ImportOntology](../RefactoringActions/ImportOntology)

## Position in class hierarchy

`owl:Thing` → `rdfs:Class` → `prov:Activity` → `fract:RefactoringAction` → `fract:GlobalAction`
